# 📸 Bulk Image Watermark Tool

[![Deployed on Netlify](https://api.netlify.com/api/v1/badges/7f815f19-9df9-4e43-a08c-3823e4c17ff7/deploy-status)](https://app.netlify.com/projects/bulk-watermark-tool/deploys)
![MIT License](https://img.shields.io/badge/License-MIT-blue?style=flat-square)

Add watermarks (text or logos) to multiple images at once. No installation required - works directly in your browser.

## Features

- Text or logo watermarks
- Bulk processing with live preview
- 9-point positioning or tiled patterns
- Supports JPEG, PNG, GIF, WebP (up to 10MB each)
- Lossless PNG output, maximum quality for JPEG/WebP
- Download individually or as ZIP

## Quick Start

1. Upload images (drag & drop or click)
2. Choose text or logo watermark
3. Adjust position, size, opacity, rotation
4. Process and download

## Perfect For

- Photographers protecting their work
- Businesses branding images
- Content creators marking their content

## Requirements

Modern browser with JavaScript enabled (Chrome 80+, Firefox 75+, Safari 13+, Edge 80+)
